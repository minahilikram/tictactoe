FORTRAN Tic-Tac-Toe
===================


************
Compilation
************
    - open terminal

    RUN:
        - type "make"


********************
Running the program
********************
    RUN:
        - type "make run"


********************
Cleaning the files
********************
    RUN:
        - type "make clean"


*****************
Known Limitations
*****************
    - On user input prompt, a number between 1 and 9 must be entered; pressing simply "enter" doesn't prompt user of any indication that the input is incorrect. Any other input is tested, user is prompted.
